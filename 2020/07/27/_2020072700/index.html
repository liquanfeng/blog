Download video from 'blob: https'

liquanfeng   in Guangdong
Mon Jul 27 2020
  • Others

今天想要从学校网站下课程视频, 学校网站不支持下载, 那尝试看看网页元素, 能直接拿到视频地址就好了, 结果发现长这样

不懂前端只能google, ffmpeg? m3u8? 尝试各种插件没找到能用的, 只能自己动手

  1. 首先找到m3u8地址, 打开视频网址, F12, 如下图找到该地址

    打开这个文件, 长这样

  2. 用ffmpeg下载这个m3u8

    ffmpeg -i 202003112241412507766playlist.m3u8 -c copy video.mp4

    • Protocol 'https' not on whitelist 'file,crypto,data'!

      加上 -protocol_whitelist "file,https,crypto,tls,tcp"

    • Unable to open key file https://vo1.xxx/**/***.key

      把这个key单独下下来, 在m3u8改相应的url

    • Unable to open resource: 202003190208447124436playlist0.ts

      如下图从网页找到这些ts的地址, 或者直接把这些ts都下下来, 然后在m3u8把这些ts的位置改成对应的位置

  3. 最后再用ffmpeg下载这个m3u8

    ffmpeg -protocol_whitelist "file,https,crypto,tls,tcp" -i .\202003190208447124436playlist.m3u8 -c copy video.mp4

    Filename extension of '6f5049bd874217e526014991.key' is not a common multimedia extension, blocked for security reasons.

    把key文件后缀改成m3u8,再在m3u8改key名字

    最后m3u8文件变成这样

    最后再再用ffmpeg下载这个m3u8, 最后生成video.mp4!
